Este taller esta dirigido a personas con o sin experiencia en desarrollo. El contenido abarca desde los conceptos mas básicos de HTML5+CSS3 hasta las características mas recientes que permitirán a los participantes crear sitios web que cumplan con los estándares semánticos para lectores de pantallas, motores de búsqueda aprovechando al máximo las características que ofrecen las hojas de estilo.

Contenido

El taller abarcará los siguientes temas:
  1. Introducción a las páginas web
  2. Fundamentos de HTML
  3. Metadata, Head
  4. Trabajando con texto dentro de un HTML
  5. Introducción a CSS
  6. Trabajando con Imágenes
  7. Hyperlinks
  8. Listas
  9. Elementos estructurales, semánticos y de contexto
  10. Audio, Video, y otros Objetos
  11. HTML5 Microdata
  12. Document Outlines
  13. Tablas
  14. Frames
  15. Forms
  16. Selectores avanzados
  17. Herencia y especificidad
  18. Box model
  19. Trabajo avanzado con imágenes
  20. Pseudo clases y pseudo elementos
